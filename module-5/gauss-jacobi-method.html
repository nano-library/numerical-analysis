Suppose we are given the system \(A\mathbf{x} = \mathbf{b}\), which can be written equation wise as, \(i = 1, 2, \ldots, n\),
✦
\(\displaystyle\sum_{j=1}^n\) \(a_{i,j}\) \(x_{j}\) = \(b_{i}\)
In Gauss-Jacobi method we find \(x_i\) from the \(i^{th}\) equation as, \(i = 1, 2, \ldots, n\),
✦
\(x_i\) \(=\) \(1\) \(a_{i,i}\) \(\Big(\) \(b_i\) \(-\) \(\displaystyle\sum_{j=1, j\neq i}^{n}\) \(a_{i,j}x_j\) \(\Big)\)
and then form the iteration method as, for \(i = 1, 2, \ldots, n\),
✦
\(x_i^{(k+1)}\) \(=\) \(1\) \(a_{i,i}\) \(\Big(\) \(b_i\) \(-\) \(\displaystyle\sum_{j=1, j\neq i}^{n}\) \(a_{i,j}x_j^{(k)}\) \(\Big)\)
We then proceed with a initial guess \(x(0) = (x^{(0)}_0 , x^{(0)}_1 , . . . , x^{(0)}_n )^T\).
✦ Example 1 solving system of linear equations
Problem

Solve the following system of linear equations using the Gauss-Jacobi method. (Perform two iterations)

\[ 3x_1 + x_2 - x_3= 7 \\[10pt] 2x_1 - 5 x_2 + 2x_3 = -8\\[10pt] x_1 + x_2 + 10x_3 = 6 \]
Solution

The iteration formula are written as:

\[ x_1^{(k+1)} = \frac{1}{3}(7 - x_2^{(k)} + x_3^{(k)}) \] \[ x_2^{(k+1)} = \frac{1}{5}(8 + 2x_1^{(k)} + 2x_3^{(k)}) \] \[ x_3^{(k+1)} = \frac{1}{10}(6 - x_1^{(k)} - x_2^{(k)}) \]

We should start with a initial guess.
Let us set \(x^{(0)} = (0, 0, 0)^T\).
Iteration 1: We have the following updated solution

\[ x_1^{(1)} = \frac{1}{3}(7 - x_2^{(0)} + x_3^{(0)}) \] \[ x_2^{(1)} = \frac{1}{5}(8 + 2x_1^{(0)} + 2x_3^{(0)}) \] \[ x_3^{(1)} = \frac{1}{10}(6 - x_1^{(0)} - x_2^{(0)}) \] \[ x_1^{(1)} = \frac{1}{3}(7 - 0 + 0) \] \[ x_2^{(1)} = \frac{1}{5}(8 + 2(0) + 2(0)) \] \[ x_3^{(1)} = \frac{1}{10}(6 - 0 - 0) \] \[ x_1^{(1)} = \frac{7}{3} = 2.333 \] \[ x_2^{(1)} = \frac{8}{5} = 1.6 \] \[ x_3^{(1)} = \frac{6}{10} = 0.6 \]

Iteration 2: We use \(x^{(1)} = (2.3333, 1.6, 0.6)^T\) to get \(x^{(2)}\)

\[ x_1^{(2)} = \frac{1}{3}(7 - 1.6 + 0.6) \] \[ x_2^{(2)} = \frac{1}{5}(8 + 2(2.3333) + 2(0.6)) \] \[ x_3^{(2)} = \frac{1}{10}(6 - 2.3333 - 1.6) \] \[ x_1^{(2)} = \frac{6}{3} = 2 \] \[ x_2^{(2)} = \frac{13.8666}{5} = 2.7733 \] \[ x_3^{(2)} = \frac{2.0667}{10} = 0.2066 \]

Therefore,

\[ x^{(2)} = (2, 2.7733, 0.2066)^T \]
✦ Example 2 solving system of linear equations
Problem

Solve the following system of linear equations using the Gauss-Jacobi method. (Perform two iterations)

\[ 4x_1 - x_2 + x_3= 8\\[10pt] 2x_1 + 5 x_2 - 3x_3 = -4\\[10pt] x_1 + x_2 + 6x_3 = 7 \]
Solution

The iteration formula are written as:

\[ x_1^{(k+1)} = \frac{1}{4}(8 + x_2^{(k)} - x_3^{(k)}) \] \[ x_2^{(k+1)} = \frac{1}{5}(-4 - 2x_1^{(k)} + 3x_3^{(k)}) \] \[ x_3^{(k+1)} = \frac{1}{6}(7 - x_1^{(k)} - x_2^{(k)}) \]

We should start with a initial guess.
Let us set \(x^{(0)} = (0, 0, 0)^T\).
Iteration 1: We have the following updated solution

\[ x_1^{(1)} = \frac{1}{4}(8 + x_2^{(0)} - x_3^{(0)}) \] \[ x_2^{(1)} = \frac{1}{5}(-4 - 2x_1^{(0)} + 3x_3^{(0)}) \] \[ x_3^{(1)} = \frac{1}{6}(7 - x_1^{(0)} - x_2^{(0)}) \]

Substituting the initial guess values,

\[ x_1^{(1)} = \frac{1}{4}(8 + 0 - 0) \] \[ x_2^{(1)} = \frac{1}{5}(-4 - 2(0) + 3(0)) \] \[ x_3^{(1)} = \frac{1}{6}(7 - 0 - 0) \]

Simplifing,

\[ x_1^{(1)} = \frac{8}{4} = 2 \] \[ x_2^{(1)} = \frac{-4}{5} = -0.8 \] \[ x_3^{(1)} = \frac{7}{6} = 1.1666 \]

Iteration 2 We use \(x^{(1)} = (2, -0.2, 1.1666)^T\) to get \(x^{(2)}\)

\[ x_1^{(2)} = \frac{1}{4}(8 + x_2^{(1)} - x_3^{(1)}) \] \[ x_2^{(2)} = \frac{1}{5}(-4 - 2x_1^{(1)} + 3x_3^{(1)}) \] \[ x_3^{(2)} = \frac{1}{6}(7 - x_1^{(1)} - x_2^{(1)}) \]

Substituting the \(x^{(1)}\) values,

\[ x_1^{(2)} = \frac{1}{4}(8 + (-0.8) - 1.1666) \] \[ x_2^{(2)} = \frac{1}{5}(-4 - 2(2) + 3(1.1666)) \] \[ x_3^{(2)} = \frac{1}{6}(7 - 2 - (-0.8)) \]

Simplifing,

\[ x_1^{(1)} = \frac{6.0333}{4} = 1.5083 \] \[ x_2^{(1)} = \frac{-4.4999}{5} = -0.8999 \] \[ x_3^{(1)} = \frac{5.8}{6} = 0.9666 \]

Therefore,

\[ x^{(2)} = (1.5083, -0.8999, 0.9666)^T \]
✦ Example 3 solving system of equations
Problem

Solve the following system of equations using Jacobi iteration scheme.

\[ \begin{bmatrix} 4 & 1 & 0 & 1 \\ 1 & 4 & 1 & 0 \\ 0 & 1 & 4 & 1 \\ 1 & 0 & 1 & 4 \end{bmatrix} x = \begin{bmatrix} 2 \\ -2 \\ 2 \\ -2 \end{bmatrix} \]

Starting with \(x^{(0)} = 0\) as initial approximation, iterate 3 times.

Solution

The iteration formulas are written as:

\[ x_1^{(k+1)} = \frac{1}{4}(2 - x_2^{(k)} - x_4^{(k)}) \\[10pt] x_2^{(k+1)} = \frac{1}{4}(-2 - x_1^{(k)} - x_3^{(k)}) \\[10pt] x_3^{(k+1)} = \frac{1}{4}(2 - x_2^{(k)} - x_4^{(k)}) \\[10pt] x_4^{(k+1)} = \frac{1}{4}(-2 - x_1^{(k)} - x_3^{(k)}) \]

We should start with an initial guess.
Let us set \(x^{(0)} = (0, 0, 0, 0)^T\).
Iteration 1: We have the following updated solution

\[ x_1^{(1)} = \frac{1}{4}(2 + x_2^{(0)} - x_4^{(0)}) \\[10pt] x_2^{(1)} = \frac{1}{4}(-2 - x_1^{(0)} - x_3^{(0)}) \\[10pt] x_3^{(1)} = \frac{1}{4}(2 - x_2^{(0)} - x_4^{(0)}) \\[10pt] x_4^{(1)} = \frac{1}{4}(-2 - x_1^{(0)} - x_3^{(0)}) \]

Substituting the initial guess values,

\[ x_1^{(1)} = \frac{1}{4}(2 - 0 - 0) \\[10pt] x_2^{(1)} = \frac{1}{4}(-2 - 0 - 0) \\[10pt] x_3^{(1)} = \frac{1}{4}(2 - 0 - 0) \\[10pt] x_4^{(1)} = \frac{1}{4}(-2 - 0 - 0) \]

Simplifying,

\[ x_1^{(1)} = \frac{2}{4} = 0.5 \\[10pt] x_2^{(1)} = \frac{-2}{4} = -0.5 \\[10pt] x_3^{(1)} = \frac{2}{4} = 0.5 \\[10pt] x_4^{(1)} = \frac{-2}{4} = -0.5 \]

Iteration 2: We use \(x^{(1)} = (0.5, -0.5, 0.5, -0.5)^T\) to get \(x^{(2)}\)

\[ x_1^{(2)} = \frac{1}{4}(2 - x_2^{(1)} - x_4^{(1)}) \\[10pt] x_2^{(2)} = \frac{1}{4}(-2 - x_1^{(1)} - x_3^{(1)}) \\[10pt] x_3^{(2)} = \frac{1}{4}(2 - x_2^{(1)} - x_4^{(1)}) \\[10pt] x_4^{(2)} = \frac{1}{4}(-2 - x_1^{(1)} - x_3^{(1)}) \]

Substituting the \(x^{(1)}\) values,

\[ x_1^{(2)} = \frac{1}{4}(2 - (-0.5) - (-0.5)) = \frac{1}{4}(2 + 0.5 + 0.5) = \frac{3}{4} = 0.75 \\[10pt] x_2^{(2)} = \frac{1}{4}(-2 - 0.5 - 0.5) = \frac{1}{4}(-3) = -0.75 \\[10pt] x_3^{(2)} = \frac{1}{4}(2 - (-0.5) - (-0.5)) = 0.75 \\[10pt] x_4^{(2)} = \frac{1}{4}(-2 - 0.5 - 0.5) = -0.75 \]

Therefore,

\[ x^{(2)} = (0.75, -0.75, 0.75, -0.75)^T \]

Iteration 3: We use \(x^{(2)} = (0.75, -0.75, 0.75, -0.75)^T\)

\[ x_1^{(3)} = \frac{1}{4}(2 - (-0.75) - (-0.75)) = \frac{1}{4}(2 + 0.75 + 0.75) = \frac{3.5}{4} = 0.875 \\[10pt] x_2^{(3)} = \frac{1}{4}(-2 - 0.75 - 0.75) = \frac{-3.5}{4} = -0.875 \\[10pt] x_3^{(3)} = \frac{1}{4}(2 - (-0.75) - (-0.75)) = 0.875 \\[10pt] x_4^{(3)} = \frac{1}{4}(-2 - 0.75 - 0.75) = -0.875 \]

Therefore,

\[ x^{(3)} = (0.875, -0.875, 0.875, -0.875)^T \]
