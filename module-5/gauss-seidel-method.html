âś¨ A simple modification of the Gauss-Jacobi method leads to the Gauss-Seidel method, which is given by, for \(i = 1, 2, \ldots, n\),
âś¦ Gauss-Seidel Method
\(x_i^{(k+1)}\) \(=\) \(1\) \(a_{i,i}\) \(\Big(\) \(b_i\) \(-\) \(\displaystyle\sum_{j=1}^{i-1}\) \(a_{i,j}x_j^{(k+1)}\) \(-\) \(\displaystyle\sum_{j=i+1}^{n}\) \(a_{i,j}x_j^{(k)}\) \(\Big)\)
When calculating \(x^{(k+1)}_i\), we already have updated values for \(x^{(k+1)}_j\), \(j=1, 2,\ldots, i-1\). This method makes use of all the updatedsolutions and hence often converge faster than the Gauss-Jacobi method.
âś¦ Example 1 solving system of linear equations
Problem

Solve the following system of linear equations using the Gauss-Seidel method. (Perform two iterations)

\[ -4x_1 + x_2 + x_3= 0\\[10pt] x_1 - 4x_2 + x_3 = -5\\[10pt] x_1 + x_2 - 4x_3 = -5 \]
Solution

The iteration formula are written as:

\[ x_1^{(k+1)} = \frac{1}{4}(x_2^{(k)} + x_3^{(k)}) \] \[ x_2^{(k+1)} = \frac{1}{4}(5 + x_1^{(k)} + x_3^{(k)}) \] \[ x_3^{(k+1)} = \frac{1}{4}(5 + x_1^{(k)} + x_2^{(k)}) \]

We should start with a initial guess.
Let us set \(x^{(0)} = (0, 0, 0)^T\).
Iteration 1 We have the following updated solution

\[ x_1^{(1)} = \frac{1}{4}(x_2^{(0)} + x_3^{(0)}) \] \[ x_1^{(1)} = \frac{1}{4}(0 + 0) \] \[ x_1^{(1)} = \frac{0}{4} = 0 \] \[ x_2^{(1)} = \frac{1}{4}(5 + x_1^{(1)} + x_3^{(0)}) \] \[ x_2^{(1)} = \frac{1}{4}(5 + 0 + 0) \] \[ x_2^{(1)} = \frac{5}{4} = 1.25 \] \[ x_3^{(1)} = \frac{1}{4}(5 + x_1^{(1)} + x_2^{(1)}) \] \[ x_3^{(1)} = \frac{1}{4}(5 + 0 + 1.25) \] \[ x_3^{(1)} = \frac{6.25}{4} = 1.5625 \]

Iteration 2 We use \(x^{(1)} = (0, 1.25, 1.5625)^T\) to get \(x^{(2)}\)

\[ x_1^{(2)} = \frac{1}{4}(x_2^{(1)} + x_3^{(1)}) \] \[ x_1^{(2)} = \frac{1}{4}(1.25 + 1.5625) \] \[ x_1^{(2)} = \frac{2.8125}{4} = 0.7031 \] \[ x_2^{(1)} = \frac{1}{4}(5 + x_1^{(2)} + x_3^{(1)}) \] \[ x_2^{(2)} = \frac{1}{4}(5 + 0.7031 + 1.5625) \] \[ x_2^{(2)} = \frac{7.2556}{4} = 1.8164 \] \[ x_3^{(1)} = \frac{1}{4}(5 + x_1^{(2)} + x_2^{(2)}) \] \[ x_3^{(2)} = \frac{1}{4}(5 + 0.7031 + 1.8164) \] \[ x_3^{(2)} = \frac{7.5195}{4} = 1.0742 \]

Therefore,

\[ x^{(2)} = (0.7031, 1.8164, 1.0742)^T \]
âś¦ Example 2 solving system of linear equations
Problem

Solve the following system of linear equations using the Gauss-Seidel method. (Perform two iterations)

\[ 5x_1 - 2x_2 + 3x_3= 7\\[10pt] -3x_1 + 9x_2 + x_3 = -8\\[10pt] 2x_1 - x_2 - 7x_3 = 6 \]
Solution

The iteration formula are written as:

\[ x_1^{(k+1)} = \frac{1}{5}(7 + 2x_2^{(k)} - 3x_3^{(k)}) \] \[ x_2^{(k+1)} = \frac{1}{9}(-8 + 3x_1^{(k)} - x_3^{(k)}) \] \[ x_3^{(k+1)} = \frac{1}{7}(-6 + 2x_1^{(k)} - x_2^{(k)}) \]

We should start with a initial guess.
Let us set \(x^{(0)} = (0, 0, 0)^T\).
Iteration 1 We have the following updated solution

\[ x_1^{(1)} = \frac{1}{5}(7 + 2x_2^{(0)} - 3x_3^{(0)}) \] \[ x_1^{(1)} = \frac{1}{5}(7 + 2(0) - 3(0)) \] \[ x_1^{(1)} = \frac{7}{5} = 1.4 \] \[ x_2^{(1)} = \frac{1}{9}(-8 + 3x_1^{(1)} - x_3^{(0)}) \] \[ x_2^{(1)} = \frac{1}{9}(-8 + 3(1.4) - 0) \] \[ x_2^{(1)} = \frac{-3.8}{9} = -0.4222 \] \[ x_3^{(1)} = \frac{1}{7}(-6 + 2x_1^{(1)} - x_2^{(1)}) \] \[ x_3^{(1)} = \frac{1}{7}(-6 + 2(1.4) - (-0.4222)) \] \[ x_3^{(1)} = \frac{-2.7778}{7} = -0.3968 \]

Iteration 2 We use \(x^{(1)} = (1.4, -0.4222, -0.3968)^T\) to get \(x^{(2)}\)

\[ x_1^{(2)} = \frac{1}{5}(7 + 2x_2^{(1)} - 3x_3^{(1)}) \] \[ x_1^{(2)} = \frac{1}{5}(7 + 2(1.4) -3 (-0.3968)) \] \[ x_1^{(2)} = \frac{7.3460}{6} = 1.4692 \] \[ x_2^{(1)} = \frac{1}{9}(-8 + 3(x_1^{(2)}) - x_3^{(1)}) \] \[ x_2^{(2)} = \frac{1}{9}(-8 + 3(1.4692) - (-0.3968)) \] \[ x_2^{(2)} = \frac{-3.1956}{9} = -0.3550 \] \[ x_3^{(1)} = \frac{1}{7}(-6 + 2x_1^{(2)} - x_2^{(2)}) \] \[ x_3^{(2)} = \frac{1}{7}(-6 + 2(1.4692) - (-0.3550)) \] \[ x_3^{(2)} = \frac{-2.7066}{7} = -0.3866 \]

Therefore,

\[ x^{(2)} = (1.4692, -0.3550, -0.3866)^T \]
âś¦ Example 3 solving system of linear equations
Problem

Consider the system of equations,

\[ 2x-y = 1\\[10pt] -x + 2y - z = 0\\[10pt] -y - 2z - w = 0\\[10pt] -z + 2w = 1 \]

Apply Gauss-Seidel iteration scheme to the system. Starting with \(x^{(0)} = 0\) as initial approximation, iterate 3 times.

Solution

The iteration formula are written as:

\[ x^{(k+1)} = \frac{1+y^{(k)}}{2}\\[10pt] y^{(k+1)} = \frac{x^{(k)}+z^{(k)}}{2}\\[10pt] z^{(k+1)} = \frac{-y^{(k)}-w^{(k)}}{2} \\[10pt] w^{(k+1)} = \frac{1 + z^{(k)}}{2} \]

We should start with a initial guess.
Let us set \(x^{(0)} = (0, 0, 0, 0)^T\).
Iteration 1 We have the following updated solution

\[ x^{(1)} = \frac{1+y^{(0)}}{2} \\[10pt] x^{(1)} = \frac{1+0}{2} \\[10pt] x^{(1)} = \frac{1}{2} = 0.500 \] \[ y^{(1)} = \frac{x^{(1)}+z^{(0)}}{2} \\[10pt] y^{(1)} = \frac{0.5+0}{2} \\[10pt] y^{(1)} = \frac{0.5}{2} = 0.2500 \] \[ z^{(1)} = \frac{-y^{(1)}-w^{(0)}}{2} \\[10pt] z^{(1)} = \frac{-0.25-0}{2} \\[10pt] z^{(1)} = \frac{-0.25}{2} = -0.1250 \] \[ w^{(1)} = \frac{1 + z^{(1)}}{2} \\[10pt] w^{(1)} = \frac{1 - 0.125}{2} \\[10pt] w^{(1)} = \frac{0.875}{2} = 0.4375 \]

Iteration 2 We use \((x^{(1)}, y^{(1)}, z^{(1)}, w^{(1)}) = (0.5, 0.25, -0.125, 0.4375)^T\) to get \((x^{(2)}, y^{(2)}, z^{(2)}, w^{(2)})\)

\[ x^{(2)} = \frac{1+y^{(1)}}{2} \\[10pt] x^{(2)} = \frac{1+0.25}{2} \\[10pt] x^{(2)} = \frac{1.25}{2} = 0.6250 \] \[ y^{(2)} = \frac{x^{(2)}+z^{(1)}}{2} \\[10pt] y^{(2)} = \frac{0.625-0.125}{2} \\[10pt] y^{(2)} = \frac{0.5}{2} = 0.2500 \] \[ z^{(2)} = \frac{-y^{(2)}-w^{(1)}}{2} \\[10pt] z^{(2)} = \frac{-0.25-0.4375}{2} \\[10pt] z^{(2)} = \frac{-0.6875}{2} = -0.3437 \] \[ w^{(2)} = \frac{1 + z^{(2)}}{2} \\[10pt] w^{(2)} = \frac{1 - 0.3437}{2} \\[10pt] w^{(2)} = \frac{0.6563}{2} = 0.3281 \]

Iteration 3 We use \((x^{(2)}, y^{(2)}, z^{(2)}, w^{(2)}) = (0.625, 0.25, -0.3437, 0.3281)^T\) to get \((x^{(3)}, y^{(3)}, z^{(3)}, w^{(3)})\)

\[ x^{(3)} = \frac{1+y^{(2)}}{2} \\[10pt] x^{(3)} = \frac{1+0.25}{2} \\[10pt] x^{(3)} = \frac{1.25}{2} = 0.6250 \] \[ y^{(3)} = \frac{x^{(3)}+z^{(2)}}{2} \\[10pt] y^{(3)} = \frac{0.625-0.3437}{2} \\[10pt] y^{(3)} = \frac{0.2813}{2} = 0.1406 \] \[ z^{(3)} = \frac{-y^{(3)}-w^{(2)}}{2} \\[10pt] z^{(3)} = \frac{-0.1406-0.3281}{2} \\[10pt] z^{(3)} = \frac{-0.4687}{2} = -0.2343 \] \[ w^{(3)} = \frac{1 + z^{(2)}}{2} \\[10pt] w^{(3)} = \frac{1 - 0.2343}{2} \\[10pt] w^{(3)} = \frac{0.7657}{2} = 0.3828 \]

Therefore,

\[ (x^{(3)}, y^{(3)}, z^{(3)}, w^{(3)}) = (0.6250, 0.1406, -0.2343, 0.3828)^T \]
âś¦ Example 4 determining the rate of convergence
Problem

Determine the rate of convergence for the Seidel method for the system:

\[ \begin{bmatrix} 4 & 0 & 2 \\ 0 & 5 & 2 \\ 5 & 4 & 10 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \\ x_3 \end{bmatrix} = \begin{bmatrix} 4 \\ -3 \\ 2 \end{bmatrix} \]
Solution

We now determine the rate of convergence for the Gauss-Seidel method for the system:

\[ \begin{bmatrix} 4 & 0 & 2 \\ 0 & 5 & 2 \\ 5 & 4 & 10 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \\ x_3 \end{bmatrix} = \begin{bmatrix} 4 \\ -3 \\ 2 \end{bmatrix} \]

The Gauss-Seidel iteration matrix is \(B_{GS} = -(D+L)^{-1} U\), where \(A = D + L + U\) with \(D\) diagonal, \(L\) strictly lower triangular, \(U\) strictly upper triangular.

From \(A\):

\[ D = \begin{bmatrix} 4 & 0 & 0 \\ 0 & 5 & 0 \\ 0 & 0 & 10 \end{bmatrix} \] \[ L = \begin{bmatrix} 0 & 0 & 0 \\ 0 & 0 & 0 \\ 5 & 4 & 0 \end{bmatrix} \] \[ U = \begin{bmatrix} 0 & 0 & 2 \\ 0 & 0 & 2 \\ 0 & 0 & 0 \end{bmatrix} \]

Form \(D+L\):

\[ D+L = \begin{bmatrix} 4 & 0 & 0 \\ 0 & 5 & 0 \\ 5 & 4 & 10 \end{bmatrix} \]

Find \((D+L)^{-1}\). Solve \((D+L)X = I\) for the inverse by rows.

Let \[ (D+L)^{-1} = \begin{bmatrix} a & b & c \\ d & e & f \\ g & h & i \end{bmatrix} \].

First column of inverse solves \( (D+L) \begin{bmatrix} a \\ d \\ g \end{bmatrix} = \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix} \):

\[ 4a = 1 \implies a = 1/4 \] \[ 0a + 5d = 0 \implies d = 0 \] \[ 5a + 4d + 10g = 0 \implies 5(1/4) + 0 + 10g = 0 \implies 10g = -5/4 \implies g = -1/8 \]

Second column solves \( (D+L) \begin{bmatrix} b \\ e \\ h \end{bmatrix} = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} \):

\[ 4b = 0 \implies b = 0 \] \[ 0b + 5e = 1 \implies e = 1/5 \] \[ 5b + 4e + 10h = 0 \implies 0 + 4(1/5) + 10h = 0 \implies 10h = -4/5 \implies h = -2/25 = -0.08 \]

Third column solves \( (D+L) \begin{bmatrix} c \\ f \\ i \end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ 1 \end{bmatrix} \):

\[ 4c = 0 \implies c = 0 \] \[ 0c + 5f = 0 \implies f = 0 \] \[ 5c + 4f + 10i = 1 \implies 0 + 0 + 10i = 1 \implies i = 1/10 \]

Thus:

\[ (D+L)^{-1} = \begin{bmatrix} 1/4 & 0 & 0 \\ 0 & 1/5 & 0 \\ -1/8 & -2/25 & 1/10 \end{bmatrix} \]

Now \(U\):

\[ U = \begin{bmatrix} 0 & 0 & 2 \\ 0 & 0 & 2 \\ 0 & 0 & 0 \end{bmatrix} \]

Gauss-Seidel iteration matrix: \(B_{GS} = -(D+L)^{-1} U\).

Compute \((D+L)^{-1} U\):

\[ M = \begin{bmatrix} 1/4 & 0 & 0 \\ 0 & 1/5 & 0 \\ -1/8 & -2/25 & 1/10 \end{bmatrix} \begin{bmatrix} 0 & 0 & 2 \\ 0 & 0 & 2 \\ 0 & 0 & 0 \end{bmatrix} \]

First column of product: all zeros because first column of \(U\) is zeros.

Second column of product: all zeros because second column of \(U\) is zeros.

Third column: \(U\)'s third column = \([2, 2, 0]^T\). Multiply \(M \cdot [2,2,0]^T\):

\[ (1/4)(2) + 0 + 0 = 0.5 \] \[ 0 + (1/5)(2) + 0 = 0.4 \] \[ (-1/8)(2) + (-2/25)(2) + 0 = -0.25 - 0.16 = -0.41 \]

So:

\[ (D+L)^{-1}U = \begin{bmatrix} 0 & 0 & 0.5 \\ 0 & 0 & 0.4 \\ 0 & 0 & -0.41 \end{bmatrix} \]

Thus:

\[ B_{GS} = - \begin{bmatrix} 0 & 0 & 0.5 \\ 0 & 0 & 0.4 \\ 0 & 0 & -0.41 \end{bmatrix} = \begin{bmatrix} 0 & 0 & -0.5 \\ 0 & 0 & -0.4 \\ 0 & 0 & 0.41 \end{bmatrix} \]

Eigenvalues of \(B_{GS}\): This matrix is upper triangular, so eigenvalues are the diagonal entries: \(0, 0, 0.41\).

Spectral radius:

\[ \rho(B_{GS}) = 0.41 \]

Rate of convergence (asymptotic):

\[ R_{GS} = -\ln(\rho(B_{GS})) = -\ln(0.41) \approx 0.8916 \]

Final answer:

\[ \rho(B_{GS}) = 0.41,\quad R_{GS} \approx 0.8916 \]

The Gauss-Seidel method converges faster than the Jacobi method here because \(\rho(B_{GS}) < \rho(B_J)\).
