Given a system \(Ax = b\), it can be written as, where \(Q\) is an invertible matrix,
✦
\(Q\mathbf{x}\) \(=\) \(Q\mathbf{x}\) \(-\) \(A\mathbf{x}\) \(+\) \(\mathbf{b}\)
✦
\(\mathbf{x}\) \(=\) \((I\) \(-\) \(Q^{-1}A)\) \(\mathbf{x}\) \(+\) \(Q^{-1}\mathbf{b}\)
Iterative method can be defined as, for \(k = 0, 1, 2, \ldots\),
✦
\(\mathbf{x}^{(k+1)}\) \(=\) \((I\) \(-\) \(Q^{-1}A)\) \(\mathbf{x}^{(k)}\) \(+\) \(Q^{-1}\mathbf{b}\)
We can find the error as
✦
\(\|\mathbf{x}^{(k+1)} - \mathbf{x}\|\) \(=\) \(\|(I - Q^{-1}A)(\mathbf{x}^{(k)} - \mathbf{x})\|\)
✦
\(\|\mathbf{x}^{(k+1)} - \mathbf{x}\|\) \(\leq\) \(\|I - Q^{-1}A\|\) \(\|\mathbf{x}^{(k)} - \mathbf{x}\|\)
✦
\(\|\mathbf{x}^{(k+1)} - \mathbf{x}\|\) \(\leq\) \(\|I - Q^{-1}A\|^2\) \(\|\mathbf{x}^{(k-1)} - \mathbf{x}\|\)
✦
\(\|\mathbf{x}^{(k+1)} - \mathbf{x}\|\) \(\leq\) \(\|I - Q^{-1}A\|^{k+1}\) \(\|\mathbf{x}^{(0)} - \mathbf{x}\|\)
✦
\(\|\mathbf{x}^{(k+1)} - \mathbf{x}\|\) \(\rightarrow\) \(0\) when \(\|I - Q^{-1}A\|\) \(<\) \(1\)
✦ Iteration Matrix
✦
\(H\) \(=\) \((I\) \(-\) \(Q^{-1}A)\)
Hence, if \(\|H\| < 1\), the iteration method converges.
The necessary and sufficient condition for the convergence of an iterative method of the form, for \(k = 0, 1, 2, 3, \ldots\),
✦
\(\mathbf{x}^{(k+1)}\) \(=\) \(H\mathbf{x}^{(k)}\) \(+\) \(\mathbf{c},\)
is that the eigenvalues of the iteration matrix satisfy \(|\lambda_i(H)| < 1\), \(i = 1, 2, 3, \ldots , n\) or simply, \(\rho(H) < 1\).
Suppose, \(A = L + D + U\) where \(L\), \(U\), and \(D\) contain the lower, upper and diagonal parts of \(A\) respectively. Then, the Jacobi method can be obtained from Equation (21.3) by using \(Q = D\), whereas the Gauss-Seidel method can be obtained using \(Q = L + D\).
If \(A\) is strictly diagonally dominant, then the sequence of approximations generated by Jacobi method to solve \(Ax = b\) converges.
