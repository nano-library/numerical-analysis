✨ Vector norm provides a way to quantify the size or length of a vector, and can be used to measure the distance between vectors.
Let \(\mathbb{R}^n\) denote the set of all n-dimensional column vectors with real-number components. To define a distance in \(\mathbb{R}^n\) we use the notion of a norm, which is the generalization of the absolute value on \(\mathbb{R}\), the set of real numbers.
🌸
Vector Norm

A vector norm on \(\mathbb{R}^n\) is a function, \(\| \cdot \|\), from \(\mathbb{R}^n\) into \(\mathbb{R}\) with the following properties:

  • Positivity:
    \(\|x\| \geq 0\) for all \(x \in \mathbb{R}^n\)
  • \(\|x\| = 0\) if and only if \(x = 0\)
  • Homogeneity:
    \(\|\alpha x\| = |\alpha| \; \|x\|\) for \(\alpha \in \mathbb{R}\) and \(x \in \mathbb{R}^n\)
  • Triangle inequality:
    \(\|x + y\| \leq \|x\| + \|y\|\) for \(x, y \in \mathbb{R}^n\)
Vectors in \(\mathbb{R}^n\) are column vectors, and it is convenient to use the transpose notation when a vector is represented in terms of its components. For example, the vector
✦
\(x\) = x1 x2 \(\vdots\) xn
will be written \(x = \left [x_1 \; x_2 \; \ldots \; x_n \right]^T\)
💠
Note
Each of these norms reduces to the absolute value in the case \(n = 1\).

\(l^1\) norm

Given a vector \(x = \left [x_1 \; x_2 \; \ldots \; x_n \right]^T\) of size \(n\), the \(l^1\) norm of \(x\) is defined as:
✦
\(\|x\|_1\) = \(\displaystyle\sum_{i=1}^n\) \(|x_i|\)

Euclidean (or \(l^2\)) norm

The \(l^2\) norm is called the Euclidean norm of the vector \(x\) because it represents the usual notion of distance from the origin in case \(x\) is in \(\mathbb{R}^1 \equiv \mathbb{R}, \mathbb{R}^2, \text{or } \mathbb{R}^3\). For example, the \(l^2\) norm of the vector \(x = \left [x_1 ,\; x_2 ,\; x_3 \right]^T\) gives the length of the straight line joining the points \((0, 0, 0)\) and \((x_1, x_2, x_3)\).
Given a vector \(x = \left [x_1 \; x_2 \; \ldots \; x_n \right]^T\) of size \(n\), the Euclidean norm of \(x\) is defined as:
✦
\(\|x\|_2\) = \(\sqrt{\displaystyle\sum_{i=1}^n \color{#6AAEE0}{x_i^2}}\)

Maximum (or \(l^{\infty}\)) norm

Given a vector \(x = \left [x_1 \; x_2 \; \ldots \; x_n \right]^T\) of size \(n\), the Maximum norm of \(x\) is defined as:
✦
\(\|x\|_{\infty}\) = \(\displaystyle\max_{1 \leq i \leq n} \color{#6AAEE0}{|x_i|}\)
✦ Example 1 finding norm of the vector
Problem
Find \(\|·\|_1\), \(\|·\|_2\) and \(\|·\|_{\infty}\) norm of the vector \(x = \left [1 ,\; 1 ,\; 2 ,\; 0 ,\; 3 \right]^T\).
Solution

We will compute the vector norms of

\[ x = \left[1,\;1,\;2,\;0,\;3\right]^T \]

The definitions of norms:

\[ \|x\|_1 = \sum_{i=1}^{n} |x_i|, \] \[ \|x\|_2 = \sqrt{\left(\sum_{i=1}^{n} |x_i|^2\right)}, \] \[ \|x\|_{\infty} = \max_{1 \le i \le n} |x_i| \]

First, we compute the \( \|x\|_1 \) norm

\[ \|x\|_1 = |1| + |1| + |2| + |0| + |3| \] \[ \|x\|_1 = 1 + 1 + 2 + 0 + 3 = 7 \]

Next, we compute the \( \|x\|_2 \) norm

\[ \|x\|_2 = \sqrt{\left(1^2 + 1^2 + 2^2 + 0^2 + 3^2\right)} \] \[ \|x\|_2 = \sqrt{\left(1 + 1 + 4 + 0 + 9\right)} \] \[ \|x\|_2 = \sqrt{15} \]

Finally, we compute the \( \|x\|_{\infty} \) norm

\[ \|x\|_{\infty} = \max\{ |1|, |1|, |2|, |0|, |3| \} \] \[ \|x\|_{\infty} = 3 \]

Hence, the norms of the vector are

\[ \|x\|_1 = 7,\qquad \|x\|_2 = \sqrt{15},\qquad \|x\|_{\infty} = 3 \]
✦ Example 2 finding norm of the vector
Problem
Determine the \(l^2\) norm and the \(l^{\infty}\) norm of the vector \(x = \left [-1 ,\; 1 ,\; -2 \right]^T\).
Solution

We will compute the vector norms of

\[ x = \left [-1 ,\; 1 ,\; -2 \right]^T \]

The definitions of norms:

\[ \|x\|_2 = \sqrt{\left(\sum_{i=1}^{n} |x_i|^2\right)}, \] \[ \|x\|_{\infty} = \max_{1 \le i \le n} |x_i| \]

First, we compute the \( \|x\|_2 \) norm

\[ \|x\|_2 = \sqrt{\left((-1)^2 + 1^2 + (-2)^2\right)} \] \[ \|x\|_2 = \sqrt{\left(1 + 1 + 4\right)} \] \[ \|x\|_2 = \sqrt{6} \]

Then, we compute the \( \|x\|_{\infty} \) norm

\[ \|x\|_{\infty} = \max\{ |-1|, |1|, |-2|\} \] \[ \|x\|_{\infty} = 2 \]

Hence, the norms of the vector are

\[ \|x\|_2 = \sqrt{6},\qquad \|x\|_{\infty} = 2 \]
