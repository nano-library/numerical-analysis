✨ A matrix norm is a norm defined on the space of matrices. Just like a vector norm assigns a scalar value to each vector, a matrix norm assigns a scalar value to each matrix.
Let \(A\) be a matrix, and let \(x\) be a vector, A matrix norm on the set of all \(n \times n\) matrices is a real-valued function, \(\|\cdot\|\), defined on this set, satisfying for all \(n \times n\) matrices A and B and all real numbers \(\alpha\).
  • Positivity:
    \(\|A\| \geq 0\), \(\|A\| = 0\) if and only if \(A = 0\).
  • Homogeneity:
    \(\|c A\| = |c|\|A\|\) for any scalar c and any matrix A.
  • Triangle inequality:
    \(\|A + B\| \leq \|A\| + \|B\|\) for any matrices A and B.
  • Submultiplicativity:
    \(\|AB\| \leq \|A\|\|B\|\) for any matrices A and B
  • Compatibility with vector norms:
    \(\|Ax\| \leq \|A\|\|x\|\) for any matrix A and any vector x.

Column Norm

Column norm of a matrix \(A\) is defined by
✦
\(\|A\|_1\) = \(\displaystyle\max_{1 \leq j \leq n}\sum_{i=1}^n \color{#6AAEE0}{|a_{i,j}|}\)

Frobenius Norm

Given a matrix \(A\), the Frobenius norm is given by
✦
\(\|A\|_F\) = \(\sqrt{\displaystyle\sum_{i=1}^n \sum_{j=1}^n \color{#6AAEE0}{|a_{i,j}|^2}}\)

Row Norm

Row norm of A is defined by
✦
\(\|A\|_{\infty}\) = \(\displaystyle\max_{1 \leq i \leq n}\sum_{j=1}^n \color{#6AAEE0}{|a_{i,j}|}\)

Spectral Norm

The spectral norm \((\|A\|_2)\) of a matrix \(A\) is defined as the square root of the largest eigenvalue of the symmetric matrix \(A^TA\), where \(\lambda_{max}(A^TA)\) is the largest eigenvalue of \(A^TA\),
✦
\(\|A\|_2\) = \(\sqrt{\lambda_{max}(A^TA)}\)

Operator Norm (Matrix Norms Induced by Vector Norms)

Given a vector norm \(\|\cdot\|_v\) the matrix norm induced by the vector norm \(\|\cdot\|_v\) is defined by
✦
\(\|A\|_v\) = \(\displaystyle\sup_{x \neq 0}\) \(\|A\|_v\) \(\|x\|_v\)
Equivalently, one can also define the following:
✦
\(\|A\|_v\) = \(\displaystyle\sup_{\|x\| = 0} \|Ax\|_v\)
✦ Example 1 finding norm of the matrix
Problem

Find \(\|\cdot\|_1\) , \(\|\cdot\|_F\) and \(\|\cdot\|_{\infty}\) norm of the following matrix

\[ A = \begin{bmatrix} 3 & 8 & 9 & 7\\ 6 & 2 & 4 & 5\\ 9 & 7 & 6 & 3\\ 8 & 1 & 5 & 2 \end{bmatrix} \]
Solution

\(\|\cdot\|_1\) norm (maximum absolute column sum)

\[ |A\|_1 = \max_{1 \leq j \leq n} \sum_{i=1}^n |a_{i,j}| \] \[ \text{Column 1: } 3+6+9+8=26\\ \text{Column 2: } 8+2+7+1=18\\ \text{Column 3: } 9+4+6+5=24\\ \text{Column 4: } 7+5+3+2=17 \] \[ \|A\|_1 = 26 \]

\(\|\cdot\|_F\) norm (Frobenius norm)

\[ \|A\|_F = \sqrt{\sum_{i=1}^n \sum_{j=1}^n |a_{i,j}|^2} \] \[ \text{Row 1: } 3^2+8^2+9^2+7^2=9+64+81+49=203\\ \text{Row 2: } 6^2+2^2+4^2+5^2=36+4+16+25=81\\ \text{Row 3: } 9^2+7^2+6^2+3^2=81+49+36+9=175\\ \text{Row 4: } 8^2+1^2+5^2+2^2=64+1+25+4=94 \]

Total sum of squares:

\[ 203+81+175+94=553 \] \[ \|A\|_F = \sqrt{553} \]

\(\|\cdot\|_{\infty}\) norm (maximum absolute row sum)

\[ \|A\|_{\infty} = \max_{1 \leq i \leq n}\sum_{j=1}^n |a_{i,j}| \] \[ \text{Row 1: } 3+8+9+7=27\\ \text{Row 2: } 6+2+4+5=17\\ \text{Row 3: } 9+7+6+3=25\\ \text{Row 4: } 8+1+5+2=16 \] \[ \|A\|_{\infty} = 27 \]
✦ Example 2 finding norm of the matrix
Problem

Determine the \(l_2\) norm of

\[ A = \begin{bmatrix} 10 & 15\\ 0 & 1\\ \end{bmatrix} \]
Solution

The definition of \(l_2\) norm:

\[ \|A\|_2 = \sqrt{\lambda_{\text{max}}(A^TA)} \]

Computing \(A^TA\)

\[ A = \begin{bmatrix} 10 & 15\\ 0 & 1\\ \end{bmatrix} \] \[ A^T = \begin{bmatrix} 10 & 0\\ 15 & 1\\ \end{bmatrix} \] \[ A^TA = \begin{bmatrix} 10 & 0\\ 15 & 1\\ \end{bmatrix} \begin{bmatrix} 10 & 15\\ 0 & 1\\ \end{bmatrix} \] \[ A^TA = \begin{bmatrix} 100 & 150\\ 150 & 226\\ \end{bmatrix} \]

Eigenvalues of \(A^TA\)

\[ |A-\lambda I| = 0 \] \[ \left| \begin{bmatrix} 100 & 150\\ 150 & 226\\ \end{bmatrix} - \begin{bmatrix} \lambda & 0\\ 0 & \lambda\\ \end{bmatrix} \right| = 0 \] \[ \left| \begin{bmatrix} 100-\lambda & 150\\ 150 & 226-\lambda\\ \end{bmatrix} \right| = 0 \] \[ (100 - \lambda)(226-\lambda)-150^2 = 0 \] \[ (100 - \lambda)(226-\lambda)-22500 = 0 \] \[ 22600 - 100\lambda - 226\lambda + \lambda^2 - 22500 = 0 \] \[ \lambda^2 - 326\lambda + 100 = 0 \] \[ \lambda = 325.6929, 0.3070 \]

Now, for \(l_2\) norm:

\[ \|A\|_2 = \sqrt{325.6929} \\[10pt] \|A\|_2 = 18.0469 \]
